About this page expand_more

This web page is built for SIT725 task 3.2p to demonstrate bootstrapping an Express app with a simple REST API and a Materialize front‑end.

The server exposes a single de>/api/books endpoint that returns book data as JSON, and the client uses JavaScript to fetch that data and show it as responsive cards with a genre filter and a slide‑up details panel.
