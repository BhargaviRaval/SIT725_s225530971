Features:
1) It adds to values.
2) After adding the values it will show the results instantly.
3) It also shows the history lists of previous additions.
4) It has unique and mordern UI.

4) Node.js + Express

How to Run the Project:
1) Firstly Install dependencies
   npm install
2) Start the server
   node server.js
3) Open the Browser and vist
   http://localhost:3000

How to use The Addition Calculator:
1) Users have to entre two values.
2) After entering the values click on Add (Upon adding anything other then numbers it will give error).
3) The results will show.
4) The same result gets saved to History. It keeps updating with every new addition.
